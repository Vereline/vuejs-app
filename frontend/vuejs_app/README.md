# vuejs_app

> A Vue.js project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```

## UI settings

```
# A graphical user interface (Vue UI) as an alternative to the command line tool
vue ui 
```
